Synthetic Populations


A simulated population representation of some geographic region built based on the true population values (e.g. population size, age distribution, ...)

Contact Matrices


  • Method of displaying distributions of contacts between ages

  • Can be used for more accurate disease modeling

    • Diseases affect children, adults, and the ederly differently

  • Can display social differences between countries and quantitatively evaluate their effect on disease transmission

Building a Contact Matrix


  • Composed of four unique matrices

    • Households (H), school (S), work (W), community (R)

  • H, S, W are based on placing synthetic people of appropriate age into each household, school, or workplace

  • R is based on the assumption of homogeneous mixing

  • Overall contact matrix (M) is a combination of H, S, W, R

Network Classification


To determine which model network most closely mimics the empirical network under investigation based upon some predefined set of network features or measures.

Benefits of Network Classification


Having a correct model helps to
  • Increase understanding of the phenomena being modeled

    • Structure dictates functions

  • Predict interactions

  • Guide experiments

Cross Scoring


  • Based on infinite customizability

  • Allows users to select the features that are most important to the question at hand

  • Over 20% more accurate than comparable methods

Cross Scoring

  1. Select network measures of interest
    • e.g. density, average degree
  2. Select model networks for comparison
    • e.g. AGV, DMC, DMR
  3. Calculate measure median value for each type of model network
    • density: AGV $=0.0017$, DMC $=0.0039$, DMR $=0.004$
  4. Calculate measure value for empirical network
    • density = 0.0035
  5. Rank and score networks based distance from empirical value
    • DMC $=1$, DMR $=2$, AGV $=3$
  6. Repeat steps 3 - 5 for all network measure and combine scores
  7. Network type with the smallest score is the best fit
